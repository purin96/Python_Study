# Python_Study

## 설치

- brew로 pyenv설치

```
brew install pyenv
```

- python3설치

```
pyenv install --list
```

```
pyenv install 3.9.1
```

```
python --version
```

## 환경변수설정

- bash 사용시

```
vi ~/.bashrc
```

```
alias python="python3"
```

```
source ~/.bashrc
```

- zsh사용시

```
vi ~/.zshrc
```

```
alias python="python3"
```

```
source ~/.zshrc
```

```
python --version
```
