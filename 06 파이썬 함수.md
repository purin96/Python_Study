```
def jumping():
    move()
    turn_left()
    move()
    turn_right()
    move()
    turn_right()
    move()
    turn_left()
 #점프 한번 하는 함수
def turn_right():
    turn_left()
    turn_left()
    turn_left()
#오른쪽으로 가는 함수
for i in range(6):
    jumping()
# 6번반복
```
